# Admin Console

The admin console is the governance and operations surface of the project.

## Key Areas

- platform overview
- users and access
- agents
- tools
- connectors
- MCP servers and tools
- prompts and workflows
- evaluations
- approvals
- audit and ops

## Demo Data

The prototype now exposes visible mock tools, workflows, approvals, and pilot feedback context so the admin overview reads like an intentional operator console instead of an empty shell.

## Prototype Vs Production

The console demonstrates governance structure and UI patterns. It should not be mistaken for a fully production-hardened control plane.
